This component is built on [Web font of BootStrap Icon 1.2+][1], so make sure to include it before bundling:

```HTML
<link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.2.1/font/bootstrap-icons.css"
/>
```

## Basic usage

`<Icon />` will render a SVG with `16px` width & height defaultly.

<Example>
    <Icon name="bootstrap" />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <Icon name="bootstrap" />
);
```

## Sizing

Assign `px` value to `width` or `height` to resize the SVG.

<Example>
    <Icon name="bootstrap" width={32} />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <Icon name="bootstrap" width={32} />
);
```

[1]: https://icons.getbootstrap.com/#icon-font
