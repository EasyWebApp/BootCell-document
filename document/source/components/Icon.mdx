## Basic usage

The easiest way to set up icon fonts for use in any web page is through [JSDelivr CDN][1].
All you need to do is include a single line of HTML:

```HTML
<link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/@fortawesome/fontawesome-free@5.12.1/css/all.min.css"
/>
```

## Sizing

You can also directly style an icon's size by setting a `font-size` in your project's CSS that targets an icon or
directly in the `style` property of the JSX element referencing an icon.

### Relative

Icons inherit the `font-size` of their parent container which allow them to match any text you might use with them.
With the following properties, we can increase or decrease the size of icons relative to that inherited `font-size`.

<Example>
    <Icon name="camera" size="xs" />
    <Icon name="camera" size="sm" />
    <Icon name="camera" size="lg" />
    <Icon name="camera" size="2x" />
    <Icon name="camera" size="3x" />
    <Icon name="camera" size="5x" />
    <Icon name="camera" size="7x" />
    <Icon name="camera" size="10x" />
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <Fragment>
        <Icon name="camera" size="xs" />
        <Icon name="camera" size="sm" />
        <Icon name="camera" size="lg" />
        <Icon name="camera" size="2x" />
        <Icon name="camera" size="3x" />
        <Icon name="camera" size="5x" />
        <Icon name="camera" size="7x" />
        <Icon name="camera" size="10x" />
    </Fragment>
);
```

### Fixed Width

> Need to vertically align a series of icons, like in a list or navigation menu?
> We’ve included a fixed-width support styling for just those cases.

Add a property of `fixedWidth` on the JSX element referencing your icon to set one or more icons to the same fixed width.
This is great to use when varying icon widths (e.g. a tall but skinny icon atop a wide but short icon) would throw off vertical alignment.
For clarity in the following example, we’ve added a background color on the icon so you can see the fixed width and also bumped up the `font-size` of the parent element.

<Example>
    <ul className="list-unstyled" style={{ fontSize: '2rem' }}>
        <li>
            <Icon name="skating" fixedWidth className="bg-primary" /> Skating
        </li>
        <li>
            <Icon name="skiing" fixedWidth className="bg-info" /> Skiing
        </li>
        <li>
            <Icon name="skiing-nordic" fixedWidth className="bg-primary" />{' '}
            Nordic Skiing
        </li>
        <li>
            <Icon name="snowboarding" fixedWidth className="bg-info" />{' '}
            Snowboarding
        </li>
        <li>
            <Icon name="snowplow" fixedWidth className="bg-primary" /> Snowplow
        </li>
    </ul>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <ul className="list-unstyled" style={{ fontSize: '2rem' }}>
        <li>
            <Icon name="skating" fixedWidth className="bg-primary" /> Skating
        </li>
        <li>
            <Icon name="skiing" fixedWidth className="bg-info" /> Skiing
        </li>
        <li>
            <Icon name="skiing-nordic" fixedWidth className="bg-primary" />{' '}
            Nordic Skiing
        </li>
        <li>
            <Icon name="snowboarding" fixedWidth className="bg-info" />{' '}
            Snowboarding
        </li>
        <li>
            <Icon name="snowplow" fixedWidth className="bg-primary" />{' '}
            Snowplow

    </ul>
);
```

## Bordered + Pulled Icons

> Occasionally you might want to wrap text around an icon or give it some visual distinction with a border.
> Here’s how with the support styling we’ve included.

Use `border` and `pull="right"` or `pull="left"` for easy pull quotes or article icons.

<Example>
    <Icon name="quote-left" size={2} pull="left" />
    Gatsby believed in the green light, the orgastic future that year by year recedes
    before us. It eluded us then, but that’s no matter — tomorrow we will run faster,
    stretch our arms further... And one fine morning — So we beat on, boats against
    the current, borne back ceaselessly into the past.
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <Fragment>
        <Icon name="quote-left" size={2} pull="left" />
        Gatsby believed in the green light, the orgastic future that year by year recedes
        before us. It eluded us then, but that’s no matter — tomorrow we will run faster,
        stretch our arms further... And one fine morning — So we beat on, boats against
        the current, borne back ceaselessly into the past.
    </Fragment>
);
```

## Rotating

> Sometimes you need to rotate, flip, or mirror an icon for it to work in your project or design.
> We’ve included some quick utilities to help with that.

To arbitrarily rotate and flip icons, use the `rotate` and `flip` properties when you reference an icon.

<Example className="fa-4x">
    <Icon name="snowboarding" />
    <Icon name="snowboarding" rotate={90} />
    <Icon name="snowboarding" rotate={180} />
    <Icon name="snowboarding" rotate={270} />
    <Icon name="snowboarding" flip="horizontal" />
    <Icon name="snowboarding" flip="vertical" />
    <Icon name="snowboarding" flip="both" />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <div className="fa-4x">
        <Icon name="snowboarding" />
        <Icon name="snowboarding" rotate={90} />
        <Icon name="snowboarding" rotate={180} />
        <Icon name="snowboarding" rotate={270} />
        <Icon name="snowboarding" flip="horizontal" />
        <Icon name="snowboarding" flip="vertical" />
        <Icon name="snowboarding" flip="both" />
    </div>
);
```

## Animating

> Need a loading or status communicating-icon to spin? You’ve come to the right place.
> We’ve included some basic animations in our support styling for you to use.

Use the `animation="spin"` property to get any icon to rotate, and use `animation="pulse"` to have it rotate with 8 steps.
Works especially well with `name="spinner"` & everything in the [spinner icons category][3].

<Example className="fa-3x">
    <Icon name="spinner" animation="spin" />
    <Icon name="circle-notch" animation="spin" />
    <Icon name="sync" animation="spin" />
    <Icon name="cog" animation="spin" />
    <Icon name="spinner" animation="pulse" />
    <Icon name="stroopwafel" animation="spin" />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <div className="fa-3x">
        <Icon name="spinner" animation="spin" />
        <Icon name="circle-notch" animation="spin" />
        <Icon name="sync" animation="spin" />
        <Icon name="cog" animation="spin" />
        <Icon name="spinner" animation="pulse" />
        <Icon name="stroopwafel" animation="spin" />
    </div>
);
```

## Stacked

> You can stack our icons easily with the support styling bundled with Font Awesome.

To stack multiple icons, use the `fa-stack` class on the parent HTML element of the 2 icons you want to stack.
Then add the `stack={1}` property for the regularly sized icon and add the `stack={2}` property for the larger icon.
`inverse` can be added to the icon with the `stack={1}` to help with a knock-out looking effect.
You can even throw larger icon classes on the parent to get further control of sizing.

<Example>
    <BGIcon type="square" group="brands" name="twitter" size={2} />
    <BGIcon type="circle" name="flag" size={2} />
    <BGIcon type="square" name="terminal" size={2} />
    <BGIcon type="square" name="terminal" size={4} />
    <span className="fa-stack fa-2x">
        <Icon name="camera" stack={1} />
        <Icon name="ban" stack={2} color="danger" />
    </span>
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Icon, BGIcon } from 'boot-cell/source/Reminder/Icon';

render(
    <Fragment>
        <BGIcon type="square" group="brands" name="twitter" size={2} />
        <BGIcon type="circle" name="flag" size={2} />
        <BGIcon type="square" name="terminal" size={2} />
        <BGIcon type="square" name="terminal" size={4} />
        <span className="fa-stack fa-2x">
            <Icon name="camera" stack={1} />
            <Icon name="ban" stack={2} color="danger" />
        </span>
    </Fragment>
);
```
