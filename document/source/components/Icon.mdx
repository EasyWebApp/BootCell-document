This component is built on [SVG sprite of BootStrap Icon v1][1], so make sure to install it before bundling:

```Shell
npm install bootstrap-icons
```

## Basic usage

`<Icon />` will render a SVG with `16px` width & height defaultly.

<Example>
    <Icon name="bootstrap" />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <Icon name="bootstrap" />
);
```

## Sizing

Assign `px` value to `width` or `height` to resize the SVG.

<Example>
    <Icon name="bootstrap" width={32} />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Icon } from 'boot-cell/source/Reminder/Icon';

render(
    <Icon name="bootstrap" width={32} />
);
```

[1]: https://icons.getbootstrap.com/#sprite
