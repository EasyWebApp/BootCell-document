## Default calendar

`<WeekCalendar />` will render a 24 hours Calendar Table with 2 hours interval.

<Example>
    <WeekCalendar />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { WeekCalendar } from 'boot-cell/source/Calendar/WeekCalendar';

render(
    <WeekCalendar />
);
```

## Custom Time captions

You can custom `startTime`, `endTime` & `hourInterval` properties to change table rows.

<Example>
    <WeekCalendar startTime="09:00" endTime="22:30" hourInterval={1.5} />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { WeekCalendar } from 'boot-cell/source/Calendar/WeekCalendar';

render(
    <WeekCalendar startTime="09:00" endTime="22:30" hourInterval={1.5} />
);
```

## Custom Date cell

You can render Custom contents with `renderCell(start: Date)` function.

<Example>
    <WeekCalendar
        renderCell={start => `Class starts at ${formatDate(start, 'HH:mm')}`}
    />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { formatDate } from 'web-utility/source/date';
import { WeekCalendar } from 'boot-cell/source/Calendar/WeekCalendar';

render(
    <WeekCalendar
        renderCell={start => `Class starts at ${formatDate(start, 'HH:mm')}`}
    />
);
```
