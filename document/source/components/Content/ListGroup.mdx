## Example

Add `active` property to a `<ListItem />` to indicate the current active selection,
or `disabled` property to make it appear disabled.

<Example>
    <ListGroup>
        <ListItem active>WebCell</ListItem>
        <ListItem disabled>Cell Router</ListItem>
        <ListItem>BootCell</ListItem>
        <ListItem>KoAJAX</ListItem>
        <ListItem>MarkCell</ListItem>
    </ListGroup>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { ListGroup, ListItem } from 'boot-cell/source/Content/ListGroup';

render(
    <ListGroup>
        <ListItem active>WebCell</ListItem>
        <ListItem disabled>Cell Router</ListItem>
        <ListItem>BootCell</ListItem>
        <ListItem>KoAJAX</ListItem>
        <ListItem>MarkCell</ListItem>
    </ListGroup>
);
```

## Flush

Add `flush` property to remove some borders and rounded corners to render list group items edge-to-edge in a parent container (e.g., cards).

<Example>
    <ListGroup flush>
        {['WebCell', 'Cell Router', 'BootCell', 'KoAJAX', 'MarkCell'].map(
            text => (
                <ListItem>{text}</ListItem>
            )
        )}
    </ListGroup>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { ListGroup, ListItem } from 'boot-cell/source/Content/ListGroup';

render(
    <ListGroup flush>
        {['WebCell', 'Cell Router', 'BootCell', 'KoAJAX', 'MarkCell'].map(
            text => (
                <ListItem>{text}</ListItem>
            )
        )}
    </ListGroup>
)
```

## Horizontal

Add `horizontal` property to change the layout of list group items from vertical to horizontal across all breakpoints.
Alternatively, choose a responsive variant `sm`, `md`, `lg` & `xl` to make a list group horizontal starting at that breakpoint’s `min-width`.
Currently **horizontal list groups cannot be combined with flush list groups.**

<Example>
    <ListGroup horizontal>
        {['WebCell', 'Cell Router', 'BootCell'].map(text => (
            <ListItem>{text}</ListItem>
        ))}
    </ListGroup>
</Example>

## Contextual classes

Use contextual classes to style list items with a stateful background and color.

<Example>
    <ListGroup>
        <ListItem>WebCell</ListItem>
        {[
            'primary',
            'secondary',
            'success',
            'danger',
            'warning',
            'info',
            'light',
            'dark'
        ].map(color => (
            <ListItem color={color}>A simple {color} list group item</ListItem>
        ))}
    </ListGroup>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { ListGroup, ListItem } from 'boot-cell/source/Content/ListGroup';

render(
    <ListGroup>
        <ListItem>WebCell</ListItem>
        {[
            'primary',
            'secondary',
            'success',
            'danger',
            'warning',
            'info',
            'light',
            'dark'
        ].map(color => (
            <ListItem color={color}>A simple {color} list group item</ListItem>
        ))}
    </ListGroup>
);
```

> ##### Conveying meaning to assistive technologies
>
> Using color to add meaning only provides a visual indication,
> which will not be conveyed to users of assistive technologies – such as screen readers.
> Ensure that information denoted by the color is either obvious from the content itself (e.g. the visible text),
> or is included through alternative means, such as additional text hidden with the `.sr-only` class.

## With badges

Add badges to any list group item to show unread counts, activity, and more with the help of some [utilities][1].

<Example>
    <ListGroup>
        {['WebCell', 'Cell Router', 'BootCell'].map((text, index) => (
            <ListItem className="d-flex justify-content-between align-items-center">
                {text}
                <Badge pill>{++index}</Badge>
            </ListItem>
        ))}
    </ListGroup>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { ListGroup, ListItem } from 'boot-cell/source/Content/ListGroup';
import { Badge } from 'boot-cell/source/Reminder/Badge';

render(
    <ListGroup>
        {['WebCell', 'Cell Router', 'BootCell'].map((text, index) => (
            <ListItem className="d-flex justify-content-between align-items-center">
                {text}
                <Badge pill>{++index}</Badge>
            </ListItem>
        ))}
    </ListGroup>
);
```
