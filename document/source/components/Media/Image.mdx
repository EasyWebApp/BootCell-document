## Responsive images

`max-width: 100%;` and `height: auto;` are applied to the image so that it scales with the parent element.

<Example>
    <Image
        fluid
        src="https://tech-query.me/medias/featureimages/6.jpg"
        alt="Responsive image"
    />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Image } from 'boot-cell/source/Media/Image';

render(
    <Image
        fluid
        src="https://tech-query.me/medias/featureimages/6.jpg"
        alt="Responsive image"
    />
);
```

## Image thumbnails

In addition to our [border-radius utilities][1], you can use `thumbnail` property to give an image a rounded 1px border appearance.

<Example>
    <Image
        thumbnail
        src="https://web-cell.dev/WebCell-0.a6336415.png"
        alt="Thumbnail image"
    />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Image } from 'boot-cell/source/Media/Image';

render(
    <Image
        thumbnail
        src="https://web-cell.dev/WebCell-0.a6336415.png"
        alt="Thumbnail image"
    />
);
```

## Figure images

Anytime you need to display a piece of content — like an image with a caption, consider using a `<Image caption="" />`.

<Example>
    <Image
        caption="Figure image"
        alt="Figure image"
        src="https://tech-query.me/medias/featureimages/6.jpg"
    />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Image } from 'boot-cell/source/Media/Image';

render(
    <Image
        caption="Figure image"
        alt="Figure image"
        src="https://tech-query.me/medias/featureimages/6.jpg"
    />
);
```
