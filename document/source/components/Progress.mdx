## How it works

Progress components are built with two HTML elements, some CSS to set the width, and a few attributes.
We don’t use [the HTML5 `<progress>` element][1], ensuring you can stack progress bars, animate them, and place text labels over them.

<Example>
    {[0, 25, 50, 75, 100].map(percent => (
        <Progress percent={percent} />
    ))}
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Progress } from 'boot-cell/source/Reminder/Progress';

render([0, 25, 50, 75, 100].map(percent => (
    <Progress percent={percent} />
)));
```

## Labels

Add labels to your progress bars by setting `label` property.

<Example>
    <Progress label percent={25} />
    <Progress label={percent => percent + ' percent'} percent={25} />
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Progress } from 'boot-cell/source/Reminder/Progress';

render(
    <>
        <Progress label percent={25} />
        <Progress label={percent => percent + ' percent'} percent={25} />
    </>
);
```

## Height

We only set a `height` style value on the `<Progress />`.

<Example>
    <Progress style={{ height: '1px' }} percent={25} />
    <Progress style={{ height: '20px' }} percent={25} />
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Progress } from 'boot-cell/source/Reminder/Progress';

render(
    <>
        <Progress style={{ height: '1px' }} percent={25} />
        <Progress style={{ height: '20px' }} percent={25} />
    </>
);
```

## Backgrounds

Use `status` property values to change the appearance of individual progress bars.

<Example>
    {['success', 'info', 'warning', 'danger'].map((status, index) => (
        <Progress status={status} percent={25 * ++index} />
    ))}
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Progress } from 'boot-cell/source/Reminder/Progress';

render(['success', 'info', 'warning', 'danger'].map((status, index) => (
    <Progress status={status} percent={25 * ++index} />
)));
```

## Multiple bars

Include multiple progress bars in a progress component if you need.

<Example>
    <Progress>
        <ProgressBar percent={15} />
        <ProgressBar status="success" percent={30} />
        <ProgressBar status="info" percent={20} />
    </Progress>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Progress, ProgressBar } from 'boot-cell/source/Reminder/Progress';

render(
    <Progress>
        <ProgressBar percent={15} />
        <ProgressBar status="success" percent={30} />
        <ProgressBar status="info" percent={20} />
    </Progress>
);
```

## Striped

Add `striped` property to any `<Progress />` to apply a stripe via CSS gradient over the progress bar’s background color.

<Example>
    <Progress striped percent={10} />
    {['success', 'info', 'warning', 'danger'].map((status, index) => (
        <Progress striped status={status} percent={25 * ++index} />
    ))}
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Progress } from 'boot-cell/source/Reminder/Progress';

render(
    <>
        <Progress striped percent={10} />
        {['success', 'info', 'warning', 'danger'].map((status, index) => (
            <Progress striped status={status} percent={25 * ++index} />
        ))}
    </>
);
```

## Animated stripes

The striped gradient can also be animated.
Add `animated` property to `<Progress striped />` to animate the stripes right to left via CSS3 animations.

<Example>
    <Progress striped animated percent={75} />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Progress } from 'boot-cell/source/Reminder/Progress';

render(
    <Progress striped animated percent={75} />
);
```
