This component is built on the top of [`<BGIcon />`][1], [`@nuintun/qrcode`][2] & [Web Share API][3],
these dependencies below should be installed:

```HTML
<link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/@fortawesome/fontawesome-free@5.15.1/css/all.min.css"
/>
<script src="https://cdn.jsdelivr.net/npm/share-api-polyfill@1.0.20/dist/share-min.js"></script>
```

```Shell
npm install @nuintun/qrcode
```

## Default targets

<Example>
    <ShareBar />
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { ShareBar } from 'boot-cell/source/Navigator/ShareBar';

render(
    <ShareBar />
);
```
