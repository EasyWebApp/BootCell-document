## Tabs mode

`mode="tabs"` is built based on [Nav](components/nav).

Dynamic tabbed interfaces, as described in the
[<abbr title="Web Accessibility Initiative">WAI</abbr> <abbr title="Accessible Rich Internet Applications">ARIA</abbr> Authoring Practices][2],
require `role="tablist"`, `role="tab"`, `role="tabpanel"`, and additional `aria-` attributes in order to convey their structure,
functionality and current state to users of assistive technologies (such as screen readers).

Note that dynamic tabbed interfaces should not contain dropdown menus,
as this causes both usability and accessibility issues.
From a usability perspective, the fact that the currently displayed tab’s trigger element is not immediately visible
(as it’s inside the closed dropdown menu) can cause confusion.
From an accessibility point of view, there is currently no sensible way to map this sort of construct to a standard WAI ARIA pattern,
meaning that it cannot be easily made understandable to users of assistive technologies.

## Step mode

`mode="step"` is built based on [bs-stepper][1].

### Linear

<Example>
    <TabView linear>
        <Step icon={1}>Email</Step>
        <TabPanel>
            <form onSubmit={event => event.preventDefault()}>
                <FormField label="Email address" placeholder="Enter email" />
                <Button type="submit" color="primary">
                    Next
                </Button>
            </form>
        </TabPanel>
        <Step icon={2}>Password</Step>
        <TabPanel>
            <form onSubmit={event => event.preventDefault()}>
                <FormField
                    type="password"
                    label="Password"
                    placeholder="Password"
                />
                <Button type="reset" color="danger" className="mr-3">
                    Previous
                </Button>
                <Button type="submit" color="primary">
                    Next
                </Button>
            </form>
        </TabPanel>
        <Step icon={3}>Validate</Step>
        <TabPanel>
            <form
                className="text-center"
                onSubmit={event => event.preventDefault()}
            >
                <Button type="reset" color="danger" className="mr-3">
                    Previous
                </Button>
                <Button type="submit" color="primary">
                    Submit
                </Button>
            </form>
        </TabPanel>
    </TabView>
</Example>
