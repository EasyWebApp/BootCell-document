## Base nav

The base `<Nav />` component is built with flexbox and provide a strong foundation for building all types of navigation components.
It includes some style overrides (for working with lists), some link padding for larger hit areas, and basic disabled styling.

> The base `<Nav />` component does not include any `active` state.
> The following examples include the property, mainly to demonstrate that this particular property does not trigger any special styling.

<Example>
    <Nav>
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Nav, NavLink } from 'boot-cell/source/Navigator/Nav';

render(
    <Nav>
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
);
```

## Available styles

Change the style of `<Nav />` component with modifiers and utilities. Mix and match as needed, or build your own.

### Horizontal alignment

Change the horizontal alignment of your nav with `align` property.
By default, navs are left-aligned, but you can easily change them to center or right aligned.

Centered with `align="center"`:

<Example>
    <Nav align="center">
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Nav, NavLink } from 'boot-cell/source/Navigator/Nav';

render(
    <Nav align="center">
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
);
```

<Example>

        <NavLink active>Active</NavLink>


### Tabs

Takes the basic nav from above and adds the `itemMode="tabs"` property to generate a tabbed interface.
Use them to create tabbable regions with our [`<TabView />` component](components/tabview).

<Example>
    <Nav itemMode="tabs">
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Nav, NavLink } from 'boot-cell/source/Navigator/Nav';

render(
    <Nav itemMode="tabs">
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
);
```

### Fill and justify

Force your `<Nav />`’s contents to extend the full available width one of two modifier properties.
To proportionately fill all available space with your nav items, use `itemWidth="fill"`.
Notice that all horizontal space is occupied, but not every nav item has the same width.

<Example>
    <Nav itemMode="pills" itemWidth="fill">
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Nav, NavLink } from 'boot-cell/source/Navigator/Nav';

render(
    <Nav itemMode="pills" itemWidth="fill">
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
);
```

For equal-width elements, use `itemWidth="justified"`. All horizontal space will be occupied by nav links,
but unlike the `itemWidth="fill"` above, every nav item will be the same width.

<Example>
    <Nav itemMode="pills" itemWidth="justified">
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Nav, NavLink } from 'boot-cell/source/Navigator/Nav';

render(
    <Nav itemMode="pills" itemWidth="justified">
        <NavLink active>Active</NavLink>
        <NavLink>Link</NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
);
```

## Scrollable nav

`scrollable` mode is imported from [Offcanvas example][1] of Bootstrap.

<Example>
    <Nav scrollable>
        <NavLink>Dashboard</NavLink>
        <NavLink>
            Friends{' '}
            <Badge pill color="light" className="align-text-bottom">
                27
            </Badge>
        </NavLink>
        <NavLink>Explore</NavLink>
        <NavLink>Suggestions</NavLink>
        {[1, 2, 3, 4, 5, 6, 7, 8].map(() => (
            <NavLink>Link</NavLink>
        ))}
    </Nav>
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Nav, NavLink } from 'boot-cell/source/Navigator/Nav';
import { Badge } from 'boot-cell/source/Reminder/Badge';

render(
    <Nav scrollable>
        <NavLink>Dashboard</NavLink>
        <NavLink>
            Friends{' '}
            <Badge pill color="light" className="align-text-bottom">
                27
            </Badge>
        </NavLink>
        <NavLink>Explore</NavLink>
        <NavLink>Suggestions</NavLink>
        {[1, 2, 3, 4, 5, 6, 7, 8].map(() => (
            <NavLink>Link</NavLink>
        ))}
    </Nav>
);
```

## Using dropdowns

Add dropdown menus with `title` property & `<DropMenuItem />`s.

### Tabs with dropdowns

<Example>
    <Nav>
        <NavLink active>Active</NavLink>
        <NavLink title="Dropdown">
            <DropMenuItem>Action</DropMenuItem>
            <DropMenuItem>Another action</DropMenuItem>
            <DropMenuItem>Something else here</DropMenuItem>
            <DropMenuItem />
            <DropMenuItem>Separated link</DropMenuItem>
        </NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Nav, NavLink } from 'boot-cell/source/Navigator/Nav';
import { DropMenuItem } from 'boot-cell/source/Navigator/DropMenu';

render(
    <Nav>
        <NavLink active>Active</NavLink>
        <NavLink title="Dropdown">
            <DropMenuItem>Action</DropMenuItem>
            <DropMenuItem>Another action</DropMenuItem>
            <DropMenuItem>Something else here</DropMenuItem>
            <DropMenuItem />
            <DropMenuItem>Separated link</DropMenuItem>
        </NavLink>
        <NavLink>Link</NavLink>
        <NavLink disabled>Disabled</NavLink>
    </Nav>
);
```
