This component is built on [Web font of Bootstrap Icon 1.2+][1], so make sure to include it before bundling:

```HTML
<link
    rel="stylesheet"
    href="https://unpkg.com/bootstrap-icons@1/font/bootstrap-icons.css"
/>
```

## Basic usage

`<Icon />` will render a Web font with `16px` width & height defaultly.

<Example>
    <Icon name="bootstrap" />
</Example>

```TSX
import { DOMRenderer } from 'dom-renderer';
import { Icon } from 'boot-cell';

new DOMRenderer().render(
    <Icon name="bootstrap" />
);
```

## Sizing

Resize a Web font with `size` property.

<Example>
    <Icon name="bootstrap" size={2} />
</Example>

```TSX
import { DOMRenderer } from 'dom-renderer';
import { Icon } from 'boot-cell';

new DOMRenderer().render(
    <Icon name="bootstrap" size={2} />
);
```

[1]: https://icons.getbootstrap.com/#icon-font
