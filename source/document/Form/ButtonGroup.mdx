## Basic example

Wrap a series of buttons with `<Button />` in `<ButtonGroup />`.

<Example>
    <ButtonGroup ariaLabel="Basic example">
        <Button variant="primary">Left</Button>
        <Button variant="primary">Middle</Button>
        <Button variant="primary">Right</Button>
    </ButtonGroup>
</Example>

```tsx
import { DOMRenderer } from 'dom-renderer';
import { Button, ButtonGroup } from 'boot-cell';

new DOMRenderer().render(
    <ButtonGroup ariaLabel="Basic example">
        <Button variant="primary">Left</Button>
        <Button variant="primary">Middle</Button>
        <Button variant="primary">Right</Button>
    </ButtonGroup>
);
```

> Button groups require an appropriate `role` property and explicit label to ensure assistive technologies like screen readers identify buttons as grouped and announce them.
> Use `role="group"` for button groups or `role="toolbar"` for button toolbars.
> Then use `ariaLabel` or `ariaLabelledBy` to label them.

These classes can also be added to groups of links, as an alternative to the [`<Nav />` navigation components](#Navigator/Tabs).

<Example>
    <ButtonGroup>
        <Button href="#" variant="primary" active ariaCurrent="page">
            Active link
        </Button>
        <Button href="#" variant="primary">
            Link
        </Button>
        <Button href="#" variant="primary">
            Link
        </Button>
    </ButtonGroup>
</Example>

```tsx
import { DOMRenderer } from 'dom-renderer';
import { Button, ButtonGroup } from 'boot-cell';

new DOMRenderer().render(
    <ButtonGroup>
        <Button href="#" variant="primary" active ariaCurrent="page">

            Link
        </Button>
        <Button href="#" variant="primary">

    </ButtonGroup>
);
```

## Button toolbar

Combine sets of button groups into button toolbars for more complex components.
Use utility classes as needed to space out groups, buttons, and more.

<Example>
    <ButtonToolbar ariaLabel="Toolbar with button groups">
        <ButtonGroup className="me-2" ariaLabel="First group">
            {[1, 2, 3, 4].map(index => (
                <Button variant="primary">{index}</Button>
            ))}
        </ButtonGroup>
        <ButtonGroup className="me-2" ariaLabel="Second group">
            {[5, 6, 7].map(index => (
                <Button variant="secondary">{index}</Button>
            ))}
        </ButtonGroup>
        <ButtonGroup ariaLabel="Third group">
            <Button variant="info">8</Button>
        </ButtonGroup>
    </ButtonToolbar>
</Example>

```tsx
import { DOMRenderer } from 'dom-renderer';
import { Button, ButtonGroup, ButtonToolbar } from 'boot-cell';

new DOMRenderer().render(
    <ButtonToolbar ariaLabel="Toolbar with button groups">
        <ButtonGroup className="me-2" ariaLabel="First group">
            {[1, 2, 3, 4].map(index => (
                <Button variant="primary">{index}</Button>
            ))}
        </ButtonGroup>
        <ButtonGroup className="me-2" ariaLabel="Second group">
            {[5, 6, 7].map(index => (
                <Button variant="secondary">{index}</Button>
            ))}
        </ButtonGroup>
        <ButtonGroup ariaLabel="Third group">
            <Button variant="info">8</Button>
        </ButtonGroup>
    </ButtonToolbar>
);
```

Feel free to mix input groups with button groups in your toolbars.
Similar to the example above, you'll likely need some utilities though to space things properly.

export const numberButtons = [1, 2, 3, 4].map(index => (
    <Button variant="outline-secondary">{index}</Button>
));

<Example>
    <ButtonToolbar className="mb-3" ariaLabel="Toolbar with button groups">
        <ButtonGroup className="me-2" ariaLabel="First group">
            {numberButtons}
        </ButtonGroup>
        <InputGroup>
            <InputGroupText id="btnGroupAddon">@</InputGroupText>
            <FormControl
                placeholder="Input group example"
                ariaLabel="Input group example"
                ariaDescribedBy="btnGroupAddon"
            />
        </InputGroup>
    </ButtonToolbar>
    <ButtonToolbar
        className="justify-content-between"
        ariaLabel="Toolbar with button groups"
    >
        <ButtonGroup ariaLabel="First group">{numberButtons}</ButtonGroup>
        <InputGroup>
            <InputGroupText id="btnGroupAddon2">@</InputGroupText>
            <FormControl
                placeholder="Input group example"
                ariaLabel="Input group example"
                ariaDescribedBy="btnGroupAddon2"
            />
        </InputGroup>
    </ButtonToolbar>
</Example>

```tsx
import { DOMRenderer } from 'dom-renderer';
import {
    Button,
    ButtonGroup,
    ButtonToolbar,
    InputGroup,
    InputGroupText,
    FormControl
} from 'boot-cell';

const numberButtons = [1, 2, 3, 4].map(index => (
    <Button variant="outline-secondary">{index}</Button>
));

new DOMRenderer().render(
    <>
        <ButtonToolbar className="mb-3" ariaLabel="Toolbar with button groups">
            <ButtonGroup className="me-2" ariaLabel="First group">
                {numberButtons}
            </ButtonGroup>
            <InputGroup>
                <InputGroupText id="btnGroupAddon">@</InputGroupText>
                <FormControl
                    placeholder="Input group example"
                    ariaLabel="Input group example"
                    ariaDescribedBy="btnGroupAddon"
                />
            </InputGroup>
        </ButtonToolbar>
        <ButtonToolbar
            className="justify-content-between"
            ariaLabel="Toolbar with button groups"
        >
            <ButtonGroup ariaLabel="First group">{numberButtons}</ButtonGroup>
            <InputGroup>
                <InputGroupText id="btnGroupAddon2">@</InputGroupText>
                <FormControl
                    placeholder="Input group example"
                    ariaLabel="Input group example"
                    ariaDescribedBy="btnGroupAddon2"
                />
            </InputGroup>
        </ButtonToolbar>
    </>
);
```

## Sizing

Instead of applying button sizing classes to every button in a group,
just add `size="sm"` or `size="lg"` to each `<ButtonGroup />`,
including each one when nesting multiple groups.

<Example>
    {['lg', , 'sm'].map(size => (
        <ButtonGroup size={size} ariaLabel="Large button group">
            <Button variant="outline-primary">Left</Button>
            <Button variant="outline-primary">Middle</Button>
            <Button variant="outline-primary">Right</Button>
        </ButtonGroup>
    ))}
</Example>

```tsx
import { DOMRenderer } from 'dom-renderer';
import { Button, ButtonGroup } from 'boot-cell';

new DOMRenderer().render(
    <>
        {['lg', , 'sm'].map(size => (
            <ButtonGroup size={size} ariaLabel="Large button group">
                <Button variant="outline-primary">Left</Button>
                <Button variant="outline-primary">Middle</Button>
                <Button variant="outline-primary">Right</Button>
            </ButtonGroup>
        ))}
    </>
);
```

## Nesting

Place a `<ButtonGroup />` within another `<ButtonGroup />` when you want dropdown menus mixed with a series of buttons.

<Example>
    <ButtonGroup ariaLabel="Button group with nested dropdown">
        <Button variant="primary">1</Button>
        <Button variant="primary">2</Button>
        <DropdownButton
            className="btn-group"
            variant="primary"
            caption="Dropdown"
        >
            <DropdownItem href="#">Dropdown link</DropdownItem>
            <DropdownItem href="#">Dropdown link</DropdownItem>
        </DropdownButton>
    </ButtonGroup>
</Example>

```tsx
import { DOMRenderer } from 'dom-renderer';
import { Button, ButtonGroup, DropdownButton, DropdownItem } from 'boot-cell';

new DOMRenderer().render(
    <ButtonGroup ariaLabel="Button group with nested dropdown">
        <Button variant="primary">1</Button>
        <Button variant="primary">2</Button>
        <DropdownButton
            className="btn-group"
            variant="primary"
            caption="Dropdown"
        >
            <DropdownItem href="#">Dropdown link</DropdownItem>
            <DropdownItem href="#">Dropdown link</DropdownItem>
        </DropdownButton>
    </ButtonGroup>
);
```

## Vertical variation

Make a set of buttons appear vertically stacked rather than horizontally.
**Split button dropdowns are not supported here.**

<Example>
    <ButtonGroup vertical ariaLabel="Vertical button group">
        {[1, 2, 3, 4].map(index => (
            <Button variant="primary">Button</Button>
        ))}
    </ButtonGroup>
</Example>

```tsx
import { DOMRenderer } from 'dom-renderer';
import { Button, ButtonGroup } from 'boot-cell';

new DOMRenderer().render(
    <ButtonGroup vertical ariaLabel="Vertical button group">
        {[1, 2, 3, 4].map(index => (
            <Button variant="primary">Button</Button>
        ))}
    </ButtonGroup>
);
```

<Example>
    <ButtonGroup vertical ariaLabel="Vertical button group">
        <Button variant="primary">Button</Button>

</Example>

import { DOMRenderer } from 'dom-renderer';
